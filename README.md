## üèπ Requirements

- `react-native` >= 0.62.0
- `reanimated` >= 2
- `react-native-gesture-handler` >= 1.10.3
- `react-native-device-info` >= 8.4.8
- `typescript` >= 4.1.0 (if you use TypeScript)
- `expo` >= 41 (if you use Expo)

## üöÄ Installation

```shell
yarn add react-native-notificated
```
